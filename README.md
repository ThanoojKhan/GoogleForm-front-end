Custom Form Application
This is a custom form application developed using React, Redux, MongoDB, Node.js, and Typescript. The application allows users to create custom forms with various field types and submit their responses.

Getting Started
To get started with the application, follow the steps below:

Prerequisites
Make sure you have Node.js and npm installed on your machine.

Installation
Clone the repository to your local machine:

Copy code
git clone <repository-url>
Navigate to the project directory:

Copy code
cd custom-form-application
Install the dependencies:

Copy code
npm install
Running the Application
To run the application in development mode, use the following command:

Copy code
npm run dev
This will start both the frontend and backend servers concurrently.

Accessing the Application
Once the application is running, you can access it at http://localhost:3000 in your web browser.

Usage
Creating Forms
You can create custom forms with various field types such as textboxes, radio buttons, checkboxes, and dropdowns. All fields can be marked as required.

Submitting Responses
Users can fill out the forms created by admins and submit their responses. Validation is applied to ensure that all required fields are filled out.
